# PF2E Item Compendium (Markdown)

Each item is one `.md` file with simple key/value metadata:

```md
name: Longsword
type: weapon
price_cp: 1000
bulk: 1
icon: ⚔️
image:

# Notes
...
```

Supported `type` values: `weapon`, `armor`, `consumable`, `gear`, `treasure`, `tool`.

You can override the emoji icon by setting `image:` to a small icon URL.
